# Rogue Rings

Экшн-рогалик с видом сверху (в духе Soul Knight) — **Phaser 4 + TypeScript + Vite**.
Круглая карта от города-хаба в центре к аду на краю, слот-машина стартового ролла,
пермадеч, 5 стихий с реакциями, прокачка, боссы с эволюцией класса.

> **Этап 1 — играбельный вертикальный срез.** Реализован весь основной цикл на
> placeholder-графике (цветные примитивы). Вся числовая модель и контент вынесены
> в data-файлы по дизайн-спеке — остальной контент доливается как данные.

## Запуск

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # проверка типов + production-сборка в dist/
npm test         # юнит-тесты формул (vitest)
```

## Управление

| | |
|---|---|
| **WASD** | движение |
| **мышь** | прицел (авто-атака) |
| **Space** | рывок (i-frames) |
| **J** | навык класса |
| **K** | ульта (по заряду) |
| **H** / **1–4** | хилка / выбор хилки |
| **E** | войти в хаб (в центре карты) |
| **Tab** | меню персонажа (статы/таланты/навыки/инвентарь) |

## Что реализовано в срезе

- **Стартовый ролл (§2)** — слот-машина по лестнице редкости: класс, способность,
  оружие (+1 ступень), стихия, благословение, проклятие, реликвия, раса, потенциал.
  Детерминирован от текстового сида мира.
- **5 играбельных классов** (Воин / Следопыт / Маг / Разбойник / Пиромант) с полными
  деревьями навыков; стат-таблицы всех 30 классов — в данных.
- **6 архетипов оружия** с разными паттернами атаки (дуга / тычок / серия / снаряд /
  заряжаемый выстрел / бумеранг).
- **Стихии и реакции (§3, §34–35)** — Огонь/Лёд/Молния/Яд/Аркан, статусы (Горение,
  Заморозка, Заряд, Отравление, Разлом), реакции (Пар, Детонация, Перегрев, Паралич,
  Обморожение, Растекание, Резонанс, Усиление), правило инфузии оружие×класс.
- **Кольцо 1** — 5 типов мобов на переиспользуемых AI-архетипах (§23) + босс
  **Древо-Страж** с телеграф-атаками, фазой <50% и эволюцией класса I.
- **Прокачка (§37–38)** — XP-кривая, кап уровня из ролла, +1 очко таланта/навыка за
  уровень; общее дерево талантов (Мощь/Стойкость/Ловкость/Удача) + классовое дерево.
- **Хаб-магазин (§41)** — зелья, броня по слотам, зачарование оружия, реролл.
- **Мета** — пермадеч, экран победы/смерти, рекорды и несколько миров в localStorage.

## Структура

```
src/
  data/     числа и контент из спеки (rarity, elements, classes, weapons,
            mobs, bosses, rollPools, items, armor, skills, balance, mods, theme)
  core/     системы (rng, startRoll, stats, damage, statusEngine,
            progression, economy, save, run)
  entities/ Player, Enemy, Boss, Projectile
  scenes/   Boot, StartRoll, World, Hub, UI, Menu, End
```
